# Clip Empire — Creative System (Formats, Selection, Editing)

## Goal
Make automation produce *sticky* shorts, not spam.

---

## Format Packs (category-specific)
Each category has 3–5 templates. Templates define:
- hook style (text hook / cold open / question)
- caption style (font, emphasis rules, colors)
- pattern interrupts (zoom/punch-in cadence)
- audio chain
- outro policy (usually hard-cut; minimal CTA)

**Rule:** rotate templates; don’t let every channel look identical.

---

## Clip selection (automated scoring)
### Universal signals
- Hook strength (first 0.5–1.0s): motion + loudness delta + “attention words”
- Story shape: setup → twist → payoff
- Visual clarity: face/subject visible, not chaotic
- Novelty: avoid repeating same creator/topic too often

### Category boosts
- Finance: numbers, stakes, fear/greed language
- Business: contrarian takes, founder mistakes, strong claims
- Tech/AI: surprising demos, “unexpected output”, reactions
- True crime: suspense, unanswered questions, timeline cues
- Fitness: peak exertion, fail→payoff, transformation reveals
- Food: reveal moments, disasters, reaction energy

---

## Editing upgrades that scale
1. **Kinetic captions**
   - emphasize numbers/names/verbs
   - color accents per category
2. **Pattern interrupts every 2–4s**
   - micro zoom (2–4%), punch-ins, occasional cutaway
3. **Aggressive dead-air removal**
   - tighten pauses >200–300ms
4. **Audio chain**
   - normalize → compress → limiter

---

## Original content strategy (15%)
Original does not require filming; options:
- Original narration over b-roll/screen recordings
- Original structured series ("3 mistakes", "case in 30s")
- Original remix comps with clear narrative + overlays

Purpose: reduce reused-content suppression + build moat.

---

## QA sampling
Auto-check 1–2 clips/channel/day:
- captions readable (safe margins)
- loudness range OK
- no black frames/glitch cuts
- hook exists on frame 1

If fails → rerender with different template or discard.
